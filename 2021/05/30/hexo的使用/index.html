hexo的使用

一：安装Nodejs

node -v //查看node版本
npm -v //查看npm版本
npm install -g cnpm –registry=http://registry.npm.taobao.org //安装淘宝的cnpm 管理器
cnpm -v //查看cnpm版本

二：安装hexo

cnpm install -g hexo-cli //安装hexo框架
hexo -v //查看hexo版本

三：创建博客

mkdir blog-hexo //创建blog目录
cd blog-hexo //进入blog目录
sudo hexo init //初始化博客
hexo s //启动本地博客服务，作预览用 http://localhost:4000/ #本地访问地址
hexo n [我的第一篇文章] //创建文章
#返回blog目录
hexo clean //清理
hexo g //生成

四：部署到github

#Github创建一个新的仓库 YourGithubName.github.io（名称必须是自己的github名）
cnpm install –save hexo-deployer-git //在blog目录下安装git部署插件，如果不通过，前面加sudo。

#设置_config.yml

#回到blog目录

hexo d //部署到远端

输入github的账号和密码，账号名不区分大小写。

五：自定义主题

主题链接：https://github.com/litten/hexo-theme-yilia

按照主题链接下的操作指导进行配置。

六：Hexo Admin插件管理

安装插件

npm install --save hexo-admin

启动服务器
hexo server -d
即可在localhost:4000/admin/中编辑博文了。
